# Spotify Music Features Analysis 🎵📊

[![Tableau](https://img.shields.io/badge/View-Interactive_Dashboard-E97627?style=for-the-badge&logo=tableau&logoColor=white)](https://public.tableau.com/app/profile/elaheh.paktinat/viz/SpotifyFeaturesfinall/Dashboard1)

## 📌 Project Overview
This project is an interactive exploration of Spotify's track features (Danceability, Energy, Valence, etc.). Using **Tableau**, I visualized how different musical attributes correlate and how they vary across genres and popularity levels.

## 🛠 Tech Stack
- **Tool:** Tableau Desktop / Tableau Public
- **Data Source:** Spotify Web API (Kaggle Dataset)
- **Visualizations:** Scatter Plots, Radar Charts, and Heatmaps

## ⚙️ Key Technical Features
- **Interactive Filtering:** Users can filter by genre and popularity to see real-time shifts in music trends.
- **Correlation Analysis:** Analyzed the relationship between "Loudness" and "Energy" to identify production patterns.
- **Data Storytelling:** Designed an "Executive Dashboard" layout for quick insights into what makes a song "hit" material.
